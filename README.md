# Quran Reader

A modern, responsive, and customizable web application for reading the Holy Quran.

**Live Demo:** [https://quran-reader-beige.vercel.app/](https://quran-reader-beige.vercel.app/)

## ✨ Features

- **Read & Explore:** Browse through Surahs and read the Quran with ease.
- **Customizable Typography:** Choose your preferred Arabic font and adjust both Arabic script and translation font sizes for a comfortable reading experience.
- **Persistent Settings:** Your preferred text size and font choices are saved in your browser (`localStorage`), so they are applied every time you return.
- **Responsive Design:** Optimized for a seamless experience across desktop, tablet, and mobile devices.
- **Search Capabilities:** Quickly find Surahs or specific verses.
- **Interactive UI:** Smooth transitions, accessible sidebars, and mobile-friendly drawers.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, React 19)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:**
  - [Radix UI / shadcn/ui](https://ui.shadcn.com/)
  - [Base UI](https://base-ui.com/)
  - [Vaul](https://github.com/emilkowalski/vaul) (Drawer for mobile interactions)
  - [Lucide React](https://lucide.dev/) (Icons)
- **Data Fetching:** [TanStack React Query](https://tanstack.com/query/latest)
- **Animations:** tw-animate-css

## 🚀 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
